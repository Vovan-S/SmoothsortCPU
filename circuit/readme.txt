Подготовка тестовых массивов:
(устройство памяти описано в отчете в разделе 3.2.2)
Массив для сортировки располагается по адресам 0x00-0x4f. Младшая часть
широкого операнда имеет младший адрес. По адресу 0x70 должно быть 
записан размер массива. ВНИМАНИЕ! Запись по адресу 0x70 значения больше
40 приведет к ошибке, аппарано или программно это не проверяется.


Описание файлов:

'smoothsort.circ'   --- файл реализованного процессора в logisim.
'smoothsort_out_fix.txt' --- образ программы, необходимо загрузить в 
			элемент ОЗУ "Память для команд"
'lookup_sort.txt'   --- образ lookup table, необходимо загрузить в
			элемент ПЗУ "lookup table"
'leonard.txt'	    --- образ таблицы чисел Леонарда, необходимо 
			загрузить в элемент ПЗУ "Числа Леонарда"
'array*.txt'	    --- образ памяти для операндов, массив для 
			сортировки. Необходимо загрузить в 
			элемент ОЗУ "Память для операндов"
